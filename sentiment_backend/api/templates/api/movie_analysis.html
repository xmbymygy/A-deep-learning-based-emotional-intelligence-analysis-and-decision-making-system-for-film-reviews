系统介绍

"影语心声"基于深度学习技术构建，专注于电影评论情感分析，为用户提供精准、高效的情感识别服务

系统架构设计

数据采集层

基于requests库构建智能爬虫，从豆瓣电影热门模块采集22510条短评数据，包含完整评论内容与星级评分信息

数据预处理层

执行数据清洗（去重、异常值处理）、文本优化（分词、去停用词）、标签转换与数据平衡操作，提升数据质量

模型计算层

构建双LSTM层深度学习模型，结合Embedding层、Dropout层与全连接层，实现高精度情感分类

应用展示层

基于Web技术开发用户交互界面，实现实时情感分析与结果可视化，提供直观的数据分析体验

模型性能指标

模型准确率对比

核心模型参数配置

模型层 参数配置
Embedding层 输入维度:10000, 输出维度:256
LSTM层(第一层) 单元数:128, 返回序列:是
LSTM层(第二层) 单元数:64, 返回序列:否
Dropout层 丢弃率:0.5（防止过拟合）
全连接层 单元数:16, 激活函数:ReLU
输出层 单元数:2, 激活函数:Sigmoid

数据处理流程

数据集情感分布

数据采集

从豆瓣电影热门模块采集短评数据，包含完整评论内容和星级评分，共获取22510条原始数据

豆瓣热门电影短评采集
数据清洗与预处理

数据清洗

处理空值、重复值和异常值，移除非中文字符和特殊符号，最终得到18937条有效数据

文本优化

使用jieba分词进行中文精准分词，基于自定义停用词表移除无用词汇，提升文本数据质量

分词与去停用词
标签转换与数据平衡

数据平衡

将1-2星标记为差评(0)，3-5星标记为好评(1)，通过过采样和欠采样技术平衡数据，最终各10000条样本

文本向量化

使用Keras Tokenizer将文本转换为数值序列，通过pad_sequences统一序列长度，为模型输入做准备

文本转数值序列
